# RiderPlugins

Коллекция плагинов для JetBrains Rider, разработанных для личного использования.

> **Проекты создаются с помощью нейросети.**
> Код предоставляется как есть, без каких-либо гарантий.

---

## Использование

Плагины разрабатываются под конкретные задачи и не являются коммерческими продуктами.
Вы можете свободно использовать, изучать и адаптировать код под свои нужды.

**При использовании кода просьба указывать первоисточник:**
```
https://github.com/Odin41/RiderPlugins
```

---

## Плагины

| Плагин | Описание |
|--------|----------|
| [NamespaceConverter](./NamespaceConverter) | Преобразует блочные пространства имён C# в файловые (block-scoped → file-scoped) |
| [UnusedMethodsAnalyzer](./UnusedMethodsAnalyzer) | Находит методы C# с нулевым количеством использований по всему решению |

---

## Сборка

Каждый плагин собирается независимо через Docker:

```bat
cd UnusedMethodsAnalyzer
docker-build.bat
```

Подробности — в `README.md` внутри папки каждого плагина.
